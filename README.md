# Jarvis-AI 

Jarvis is an AI based personal voice assistant for computer, currently available for Linux. It works on command line. He can automate your daily tasks like opening spotify, searching on google or he can also tell you a joke or even he can send an email to a specific perticipant.

<a href="">Check a tutorial of how to use this bot</a>

# Getting Started 
To use Jarvis on your computer run
```py
git clone https://github.com/Bangladesh-Coding-Soldierz/Jarvis-AI #clone this repository
cd Jarvis-AI #change the directory to Jarvis-AI folder
pip3 install -r requirements.txt #install all the packages needed to use jarvis
python3 jarvis.py 
```
and you are ready to use jarvis.
